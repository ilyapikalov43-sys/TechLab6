# ГОСТ 27751-2014 «Надежность строительных конструкций»

3.1 Расчетные нагрузки должны определяться согласно СП 20.13330.
3.2 Коэффициент надежности по ответственности принимается: γn = 1,0 для нормального уровня.
4.1 Предельные состояния первой группы: потеря несущей способности.
4.2 Предельные состояния второй группы: деформации, трещиностойкость.
5.1 Срок службы зданий: I класс — 100 лет, II класс — 50 лет.
